A web page which defines all text formatting tags of HTML in tabular format.

Text Formatting Tags
Syntax Tag_Name Usage Outcome
<i> Italic This tag is used to make the text in italic format.<em> is recomended. Italic
<em> Emphasize This tag is used to make the text in italic format.<em> is recomended instead of <i>tag Emphasize(italic)
<u> Underline This tag is used to Underline the text Underline
<b> Bold This tag is used to make the text in bold. Bold
<strong> Strong This tag is used to make the text in bold.<strong> is recomended instead of <b> tag. strong
<del> Delete This tag is used to indicate the deleted text with strike line Delete
<ins> Insert This tag is used to indicate the inserted text with underline Insert
<small> small This tag is used to display the text in small size compared to default text small
<big> Big This tag is used to display the text in larger size compared to default text Big
<mark> Mark This tag is used to highlight the text with yellow color. Mark
